# simple-blog
Projet Nosql Blog
Déployer le projet : 
0. Prérequis :
nodejs et mongodb
1) Dans un terminal :
mongod --dbpath data/db --logpath data/log --smallfiles

2) Recuperer le projet
git clone 

3. Lancer le server : 
Lancer le ficher server.js 
node server.js

4. Visualiser le blog
aller à l'url : 
http://localhost:8000/
